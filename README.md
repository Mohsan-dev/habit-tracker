# 📊 Habit Tracker App

A modern and interactive Habit Tracker built using **HTML, CSS, and Vanilla JavaScript**.  
It allows users to create habits, track daily progress, maintain streaks, and manage habits with a clean UI.

---

## 🚀 Live Demo 
https://keen-twilight-6fcec7.netlify.app

---

## ✨ Features

- ➕ Add new habits dynamically
- ✔ Track daily completion (checkbox system)
- 🔥 Automatic streak calculation
- 📅 Weekly calendar navigation (Previous / Next / Today)
- ✏️ Rename habits inline (click-to-edit)
- 🗑️ Delete habits with confirmation modal
- 💾 Persistent storage using LocalStorage
- 🚫 Blocks future date checking
- 🎯 Clean responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Responsive Design)
- JavaScript (Vanilla JS)
- LocalStorage API

## 📸 Features Overview
